# Limit Order Book Prediction Project

## Goal
Predict short-term price movement from limit order book data.

## Data
LOBSTER dataset (or whatever dataset you choose)

## Inputs
- Bid prices
- Ask prices
- Bid volumes
- Ask volumes

## Target
Predict next price movement:
- Up
- Down
- Neutral

## Evaluation
- Accuracy
- F1 Score
- Backtest PnL

## Project Stages

1. Load data
2. Explore data
3. Feature engineering
4. Build baseline model
5. Evaluate
6. Backtest
