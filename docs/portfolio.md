## Reservation System

Reservations prevent traders from placing
multiple limit orders backed by the same cash
or inventory.

Without reservations:

Trader has $100

Places buy order worth $80

Places another buy order worth $80

The exchange has accepted $160 worth of orders.

This is invalid.

Reservations eliminate this issue.
